ameliorating hemianopia

a repeating 500ms stimulus, noise with cotemporal white flash, at random two to six second intervals, with a counter.

use in a dark room, on a mobile phone 3m away, set maximum volume, use phone speakers not bluetooth, set maximum brightness, turn off true tone, disable browser toolbars, turn on full screen, set up and fix eyes on another target, position the phone within the affected visual field.

a prototype apparatus, reproducing some of the configuration in the paper.
